# TDD start repo

## Options
```
npm run
  lint
    runs tslint on src folder
  lint:fix
    runs lint and fixes the issue
  compile
    compiles typescript project
  build
    runs whole suite with compile, lint  and test
  test
    runs whole test suite
  test:watch
    runs tests in TTD mode
```


## Quick Start
```
  npm install
  npm run test:watch
```
